# surasaknie.github.io

Personal project hub and GitHub Pages root for all sub-projects.

Published at <https://surasaknie.github.io/project-index/> (auto-indexed).

## Projects

- [NaoDec](https://surasaknie.github.io/naodec/) — Wiring and schematics documents for the NaoDec project.
- [Dronekyll](https://surasaknie.github.io/dronekyll/) — Product documents and rocket wireframe for the Dronekyll project.
- [Smart Vertical Hydroponics](https://surasaknie.github.io/Smart-Vertical-Hydroponics/) — Schematics for the Smart Vertical Hydroponics tower.

## How it works

Each project lives in its own repo and deploys via GitHub Project Pages. The hub at `/project-index/` is auto-generated by a GitHub Actions workflow that scans for `*.html` files on every push to `main`.

See [`pages-architecture.md`](pages-architecture.md) for full details on the multi-repo setup.
